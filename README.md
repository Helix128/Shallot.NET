# Shallot.NET
(Unofficial) Dragon Ball Legends Data API

C# port of [DBLegendsAPI](https://github.com/feijoes/DBlegendsAPI)

## Dependencies
Shallot.NET includes the following packages:
- Newtonsoft.JSON
- HTMLAgilityPack
- HtmlAgilityPack.CssSelectors.NetCore
